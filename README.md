# SBTI测试项目

## 📺 项目介绍

本项目是基于SBTI人格测试理论开发的在线测试工具，旨在为用户提供便捷、精准的人格类型评估体验。

SBTI（简化版MBTI）以经典人格理论为基础，通过一系列针对性题目，帮助用户快速认知自身人格特质、行为倾向及潜在优势，广泛适用于自我认知、职业规划、人际交往参考等场景。

⚠️ 说明：本项目原型及核心逻辑源自B站UP主 @蛆肉儿串儿，本人在原作者作品的基础上，进行了功能优化、体验升级及细节完善，尊重原作者创作成果，特此注明。

## ⚠️ 优化内容

在原项目基础上，主要进行了以下优化，提升项目的可用性、稳定性与用户体验：

1. 界面优化：重构页面布局，优化视觉设计，提升界面简洁度与美观度，适配不同设备（电脑、手机）的显示效果，解决原界面可能存在的布局错乱、视觉杂乱问题。

2. 交互优化：优化测试答题流程，增加答题进度提示、题目导航功能，减少操作步骤；优化结果展示页面，增加人格类型详细解读、特质分析，让用户更易理解测试结果。

3. 功能完善：补充测试题目校验逻辑，避免无效答题；增加测试结果保存、分享功能，方便用户留存或分享自身测试结果；修复原项目可能存在的bug（如结果计算偏差、页面卡顿等）。

4. 性能优化：优化代码结构，减少冗余代码，提升页面加载速度与答题流畅度，降低页面崩溃、卡顿的概率，提升整体使用体验。

5. 细节补充：增加项目使用说明、常见问题解答，帮助用户快速上手；优化文案表述，让题目及结果解读更通俗易懂，避免歧义。

## 项目功能

- ✅ 在线答题：提供完整的SBTI测试题目，支持按顺序答题，实时显示答题进度。

- ✅ 结果计算：根据用户答题选择，自动计算人格类型（如ISTJ、ENFP等），确保计算逻辑准确。

- ✅ 结果解读：提供详细的人格类型解读，包括核心特质、行为倾向、优势与不足、适合的场景等。

- ✅ 结果保存与分享：支持保存个人测试结果，可直接分享至社交平台。

- ✅ 适配多设备：支持电脑、手机等不同终端访问，保证答题体验一致。

## 使用方法

1. 访问项目地址（或本地启动项目），进入测试首页。

2. 阅读测试说明，点击“开始测试”进入答题页面。

3. 根据自身实际情况，选择每道题的符合选项，完成所有题目后提交。

4. 系统自动生成测试结果，可查看详细解读、保存或分享结果。
